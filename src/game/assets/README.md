# Assets Layout

This folder stores runtime assets loaded by `ResourceManager`.

## Folders
- `assets/fonts/`
- `assets/audio/`
- `assets/textures/`

## Current expected files from `AngryBalls`
- `assets/fonts/ui.ttf`
- `assets/audio/collision.wav`
- `assets/audio/bgm.ogg`

If these files are missing, the game will log warnings and fallback where possible.
